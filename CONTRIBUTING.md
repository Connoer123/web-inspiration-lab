# Contributing

This is primarily a personal learning library, but suggestions and improvements are welcome.

Each recreation should:

- Live in its own folder under `components/` or `websites/`.
- Include a README with a preview, local setup steps, attribution, and learning notes.
- Credit the source of inspiration.
- Use independently written code and original or properly licensed assets.
- Include local setup instructions and a license note when needed.
- Avoid brand confusion or claims of affiliation with the original creator.

Before opening a pull request, run the checks documented inside the recreation's own folder.
