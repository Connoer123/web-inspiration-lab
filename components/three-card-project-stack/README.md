# Three-Card Project Stack

Three project cards stick, overlap, and scale while scrolling.

![Three-card project stack](../../docs/screenshots/project-stack.webp)

- Source: `ThreeCardProjectStack.tsx` and `ThreeCardProjectStack.css`
- Dependency: `framer-motion`
- [Planned public demo](https://connoer123.github.io/web-inspiration-lab/?demo=project-stack)
- Local demo: `http://127.0.0.1:5173/web-inspiration-lab/?demo=project-stack`

The public demo becomes available after GitHub Pages finishes its first deployment.

Import the stylesheet once and pass exactly three project objects. Each object needs a unique `id` and its own card content.
